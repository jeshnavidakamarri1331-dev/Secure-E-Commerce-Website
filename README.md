# Secure-E-Commerce-Website

A secure WordPress-based e-commerce platform built using **WordPress + WooCommerce**, enhanced with strong security mechanisms such as firewall protection, malware scanning, and two-factor authentication.

## Project Overview

This project demonstrates the architecture and implementation of a secure e-commerce system. It focuses on:

* User authentication and authorization
* Product browsing and cart management
* Secure checkout and payment flow
* Server-side processing using WordPress
* Database management with MySQL
* Security integration using plugins

## Tech Stack

* **Frontend**: HTML, CSS, JavaScript
* **Backend**: WordPress
* **E-Commerce Engine**: WooCommerce
* **Database**: MySQL
* **Security**: Wordfence (Firewall, Malware Scan, 2FA)
* **Hosting**: InfinityFree / cPanel-based hosting

## Security Features

* Web Application Firewall (WAF)
* Brute Force Protection
* Rate Limiting
* Malware Scanning
* Login Security (2FA)
* HTTPS Encryption

## Workflow

1. User registers/logs in
2. User browses products
3. Adds items to cart
4. Proceeds to checkout
5. Makes payment
6. Order is stored in database
7. Confirmation is sent to user

## Setup Instructions

1. Install WordPress
2. Install and activate WooCommerce
3. Configure MySQL database
4. Install security plugins (Wordfence, WP 2FA)
5. Configure HTTPS
6. Deploy on hosting platform


## Key Components

* **Client Layer**: Web browser interface
* **Server Layer**: WordPress core + WooCommerce
* **Security Layer**: Wordfence plugin
* **Database Layer**: MySQL
